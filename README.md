# Zotero Citation Map

Zotero Citation Map adds citation and reference metrics to Zotero, provides a paper-level citation inspector, and visualizes relationships between papers in an interactive graph.

## Current feature set

- Crossref, Semantic Scholar, OpenCitations, INSPIRE-HEP, and OpenAlex providers.
- Bounded parallel refresh with provider-specific request queues and cancellation.
- Persistent citation and external-work metadata caches.
- Batch resolution of additional papers through Semantic Scholar with Crossref fallback.
- Interactive graph filters, metrics, dynamic node-size scaling, and directional edge highlighting.
- Manual citation and reference corrections stored locally without modifying provider totals.
