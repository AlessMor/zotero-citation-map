# Changelog

All notable changes to Zotero Citation Map will be documented here.

The format follows [Keep a Changelog](https://keepachangelog.com/en/1.1.0/), and releases use semantic versioning while the project remains in early development.

## [Unreleased]

### Added

- Citation and reference columns with automatic and manual refresh.
- Multi-provider citation-data support.
- Plugin-owned SQLite cache with structured outgoing references.
- Interactive citation graph with collection/tag filtering, search, configurable axes, linear/log scales, collection colors, and configurable node size/labels.

### Security

- Expanded ignore rules for secrets, local profiles, databases, and build artifacts.
- Dependency install-script allowlist and locked dependencies.
- Minimal GitHub Actions permissions and commit-SHA-pinned actions.
- Privacy, security, and third-party notices.
